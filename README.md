# TechFlow App 2.0 – Sistema de Gestão de Demandas

Projeto acadêmico desenvolvido na disciplina de Metodologias Ágeis, com aplicação prática dos frameworks Scrum e Kanban para organização e desenvolvimento de um sistema de gestão de chamados.

## Sobre o Projeto

A TechFlow App 2.0 é uma startup fictícia especializada em soluções digitais para pequenas e médias empresas. 

O sistema desenvolvido tem como objetivo centralizar chamados de suporte, organizar demandas e melhorar a comunicação entre equipe técnica e clientes.

## Objetivo do Sistema

- Cadastro de chamados
- Classificação por nível de urgência
- Listagem e acompanhamento de status
- Painel visual de acompanhamento
- Organização de tarefas por fluxo Kanban

## Metodologia Utilizada

- Scrum (Sprints, cerimônias e backlog)
- Kanban (visualização do fluxo)
- Integração utilizando Trello

## Estrutura do Fluxo no Trello

O quadro foi organizado com as seguintes listas:

- Product Backlog
- Sprint Backlog
- Em Progresso
- Em Teste
- Concluído

## Benefícios Observados

- Maior previsibilidade nas entregas
- Melhor organização do fluxo de trabalho
- Comunicação estruturada
- Redução de retrabalho
- Clareza nos critérios de aceite
